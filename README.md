# mapstruct
Exemplo utilizando MapStruct

## Métodos
Requisições para a API devem seguir os padrões:
| Método | Descrição |
|---|---|
| `GET` | Retorna informações de bike ou car. |

## Instalação

Voce pode iniciar o projeto utilizando o gradle. Na linha de comando, execute:

```bash
gradle bootRun
```
